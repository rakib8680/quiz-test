Important Questions

Question 1 : What is the difference between Map , filter , forEach and find ?

Answer:
Map: Map is an Array Function that execute every element of an array and returns a new Array that passed the Execution.
Filter: When we give and instructions into the filter function , it check every element of the selected array and returns a new array of those elements that passed the condition.
Find: Find is similar to Filter function. But when it checked the elements of an array it only returns the first element that successfully passed the condition.
ForEach: ForEach loop through every elements of an Array and it does'nt return anything.

Question 2 : What is the difference between LocalStorage , SessionStorage and Cookies?

Answer:
LocalStorage: LocalStorage collects users data and save it inside the browser that the user is using. So when the user close the browser and open the browser again later, he can see the stored data. Local storage stores data permanently.
SessionStorage: SessionStorage is similar to LocalStorage. But it does'nt store data permanently. It stores data for a limited period of time. When the user close the active tab or close the browser, the data from the SessionStorage removed instantly.
Cookies : Cookies are mainly used to track an users behaviors , likings and dislikings to show him personalised things. It stores data in a very limited amount.

Question 3 : What is Event Loop and how it works ?

Answer:
Event Loop is a high level concept in JavaScript that manages execution tasks.The Event Loop continuously checks the task queue for tasks that are ready to be executed.
When a task is ready, the Event Loop moves it to the call stack. The call stack executes the tasks.Event Loop is basically used to handle asynchronous tasks.
